Formating Tags

HTML Formatting is a process of formatting text for better look and feel. These html formatting tags are divied into two categories:

1) Physical tag
_These tags are used to provide the visual apperence to the text
2) Logical tag
_These tags are used used to add some logic or semantic value to the text

1) b tag is a physical tag, which is used to bold the text
2) strong tag is logical tag, which is tells the browser that the rext is important
3) i tag is a physical tag, which is used to make text italic
4) em tag is a logical tag, which is used to display content in italic
5) mark tag is used to highlight text
6) u tag is used to underline the text
7) tt tag is used to apper a text in teletype (mono spaced font)
8) strike tag is used to draw a strikethrough on a section of text
9) hello del tag is ysed to display the delete content
10) sup tag display the content slightly above the normal line ex: log 10
11) sub tag display the content slightly below the normal line ex: log 10
Example for both sup tag and sub tag: log 10e
12) ins tag display the content which is added ex: hi I'm Feroz Feroz Ahmed
13) small tag is used to decrease the font size by 1 unit from base font size
14) big tag used to increase the font size by one conventional unit
